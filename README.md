# Setup Steps
1. Clone the repository:
   git clone https://github.com/Jai768/Intern-Task.git
   cd Intern-Task
2. Build the Docker Image:
   docker build -t intern-task-1 .
3. Run the container:
   docker run -p 5000:5000 intern-task-1
4. Open in Browser:
   http://127.0.0.1:5000
